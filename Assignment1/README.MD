# TCP UDP
## Working

Compile Client and Server .java files using

>javac client.java

>javac server.java

Run Server in first Terminal

>java server

Run Client in second Terminal

>java client

Make sure to run Server Before Client

# RMI
## Working

Compile all files

>javac *.java

Start RMI Registry

>start rmiregistry

Run Server

>java server

Run Client

>java client
